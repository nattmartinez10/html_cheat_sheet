Common CSS Properties

In this webpage you'll be able to see common CSS Properties for future reference

CSS properties for text

Label Description Example
color This changes the color of the text color: green;
background-color Changes the background color of the text element background-color:red;
font-size Sets the size of the text font-size: 16px;
font-family Specifies the font of the text font-family: Arial, sans-serif;
font-weight Sets the thickness of the text font-weight: bold;
text-align Aligns the text horizontally text-align: center;
text-transform Controls the capitalization of text text-transform: uppercase;
line-height Sets the space between lines of text line-height: 1.5;
letter-spacing Defines the space between characters letter-spacing: 2px;
word-spacing Defines the space between words word-spacing: 5px;
text-decoration Adds decoration to text like underline or line-through text-decoration: underline;
white-space Controls how whitespace is handled white-space: nowrap;
direction Specifies the text direction direction: rtl;
